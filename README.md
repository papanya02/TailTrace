
# TailTrace - Network Activity Monitoring

## Опис

TailTrace - це інструмент для моніторингу підозрілої мережевої активності. За допомогою цього інструменту ви можете аналізувати пакунки в локальній мережі та виявляти потенційні загрози.

## Встановлення

1. Клонуйте репозиторій:
   ```bash
   git clone https://github.com/yourusername/TailTrace.git
   ```

2. Встановіть необхідні залежності:
   ```bash
   pip install .
   ```

## Використання

Запустіть команду для моніторингу трафіку:
```bash
tailtrace -t <IP_or_URL>
```
